# ☁️ AWS VPC Docker Go WebApp

    Projeto desenvolvido para a disciplina de Serviços em Nuvem.

## 📌 Objetivo

    Implementar uma aplicação Web com:

    - Front-end rodando na porta 8080
    - Back-end rodando na porta 25000
    - Comunicação entre Web e Back-end via rede privada
    - Isolamento do Back-end (não acessível publicamente)

    A aplicação consiste em um gerador de senhas fortes, onde:

    - O usuário acessa a interface web
    - Clica em um botão para gerar uma senha
    - O front-end consome uma API do back-end
    - O back-end retorna uma senha forte gerada em Go
